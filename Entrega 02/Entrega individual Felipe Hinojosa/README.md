# Documentación
###### El proceso inició con el cambio de enfoque de nuestro trabajo, por ello conseguimos en Transfermarkt esta base de datos que nos permite saber cuánto se movieron las plantillas de los equipos cada año mediante la información de cantidad de fichajes y cantidad de salidas, además de una especificación de cómo fueron esos fichajes y salidas, ya sea mediante traspasos, agencia libre u otros.
###### Al obtener los datos, analizamos una por una las variables y cómo podían ayudarnos a corroborar o desmentir nuestra hipótesis. Por supuesto que las columnas más importantes serían el total de altas y de bajas, por lo que las mantenemos. 
###### A su vez, nos percatamos que la base de datos no contempla los préstamos, por lo que decidimos eliminar ambas columnas que refieren a esto, ya que este tipo de transacción también puede categorizarse como traspaso y la investigación no busca una caracterización exhaustiva del tipo de fichajes. 
###### Por otra parte, analizamos la columna de "altas otros", que tenía todo en cero, y también la borramos. No así la columna de "Bajas otros", ya que sí existen razones fuera de la agencia libre o la venta de un jugador para justificar la salida de un jugador, sin embargo, es casi inexistente la posibilidad de traer a alguien por razones "otras" que no sean el traspaso o la agencia libre.
###### Nuestra fuente principal es Transfermarkt. La elegimos porque es una de las páginas más reconocidas y confiables si hablamos de fichajes y plantillas. El sitio está constantemente actualizándose y es el referente en cuanto a los traspasos del mundo del fútbol.
###### Respecto a las herramientos, se usó vs code studio para programar un sistema de web scraping. Este sistema extrajo los datos del sitio y los guardó en un excel, logrando llegar a nuestra base de datos. 
###### Una de las preguntas más importantes que podemos responder con esta base de datos sobre quién se mueve más en un mercado. Esta información nos permite entender cómo funcionaron las ventanas de fichajes en cada club y compararlo, cuando tengamos los datos de rendimiento, con cómo fue el año deportivo del equipo.
###### Otro punto muy relevante es poder hacer una comparación sobre qué clubes hacen menos movimientos, es decir, son más consistentes con sus plantillas a lo largo del tiempo y así entender si esto tiene alguna correlación con el rendimiento deportivo año tras año, una visión de largo plazo.
###### Una tercera pregunta que puede responder esta base de datos, es cómo se mueven los equipos chilenos. Si bien esto es algo más cualitativo, nos permite conocer cómo suelen reforzarse más, ya sea como traspaso o mediante fichajes de jugadores libres.
