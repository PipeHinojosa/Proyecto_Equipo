## Autor y publicación de los datos
El autor de la base de datos es Transfermarkt. Lo que hice yo fue traspasar los datos de forma manual a una tabla de excel.

## Contenido
La base de datos contiene la convocatoria de público de los distintos equipos de la primera división del fútbol chileno, desglosados por club y recinto, recolectados desde el año 2018 al 2025.
Es una base de datos cuantitativa y estructurada. Incluye las variables de las capacidades de los recintos, los asistentes totales, el promedio de asistencia, las veces que se llenó y el porcentaje de ocupación total.

## Pertinencia
La base de datos sirve para correlacionar distitnas estadísticas. Al cruzar estos datos con índices de violencia o medidas de Estadio Seguro, 
el reportaje puede demostrar con evidencia si una política pública específica, como la reducción de aforos, causó una caída real en la asistencia o si el público dejó de asistir por incidentes violentos específicos. 

Los datos fueron extraídos desde el portal especializado Transfermarkt, una de las fuentes de referencia global para estadísticas de fútbol.
